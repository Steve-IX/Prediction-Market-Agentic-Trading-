# ============================================================================
# SIZING STRATEGY REFERENCE
# ============================================================================
#
# PERCENTAGE - Copy a percentage of the trader's position size
#   - multiplier: 0.5 means copy 50% of their size
#   - Good for: Scaling with the trader proportionally
#
# FIXED - Use a fixed dollar amount per trade
#   - fixedAmount: Always use this exact amount (e.g., $10)
#   - Good for: Consistent position sizes regardless of trader's size
#
# ADAPTIVE - Automatically adjusts based on recent performance
#   - Increases size after winning trades
#   - Decreases size after losing trades
#   - Good for: Dynamic risk management
#
# TIERED MULTIPLIERS - Different multipliers based on trade size
#   - Small trades (<$50): Use 1.5x multiplier
#   - Medium trades ($50-$200): Use 1.0x multiplier
#   - Large trades (>$200): Use 0.5x multiplier
#   - Good for: Risk management on large trades

# ============================================================================
# RANKING PRESETS REFERENCE
# ============================================================================
#
# conservative - High win rate & consistency requirements
#   - Min trades: 20, Min win rate: 55%, Max drawdown: 40%
#   - Good for: Lower risk, steady returns
#
# aggressive - Prioritizes ROI, relaxed filters
#   - Min trades: 5, Min win rate: 40%, Max drawdown: 100%
#   - Good for: Higher risk, higher potential returns
#
# balanced - Default balanced approach
#   - Min trades: 10, Min win rate: 45%, Max drawdown: 80%
#   - Good for: General use
#
# highVolume - For traders with significant volume
#   - Min trades: 50, Min volume: $10,000
#   - Good for: Finding active, liquid traders
